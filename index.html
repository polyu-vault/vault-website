Vault
An OpenStack implementation of Parallel Database as a Service

Features

  • Run entirely on OpenStack

    • Users and administrators can provision and manage multiple parallel database instances with ease.

  • Extremely low operation cost

    • Vault is built using shared-process multi-tenancy. Different from using the virtual machine technology, Vault enables the sharing of cluster computers and database installations among clients. Operating a parallel database cloud now can save up to 80% resources (e.g., cluster computers, database licenses, electricity) without jeopardizing the security and performance.

  • Performance SLA guarantee

    • Ensuring a tenant query performance with SLA

  • Support Vertica
