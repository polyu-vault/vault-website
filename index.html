Vault
Massively Parallel Processing Database as a Service for OpenStack

What is Vault?

â€‹

A free and open-source cloud deployment of Massively Parallel Process Database (MPPDB) as a Service





With a MPPDB as a service platform provided by service provider, it allows tenant create MPPDB by naming the MPPDB and choosing the nodes quantity and flavor.




MPPDB could be provisioned quickly and connection information is returned including the connection string for the JDBC Application.

Features

  • Run entirely on OpenStack

    • Users and administrators can provision and manage multiple MPPDB instances with ease.

  • Extremely low operation cost

    • Vault is built using shared-process multi-tenancy. Different from using the virtual machine technology, Vault enables the sharing of cluster computers and database installations among clients. Operating a MPPDB cloud now can save up to 80% resources (e.g., cluster computers, database licenses, electricity) without jeopardizing the security and performance.

  • Performance SLA Guarantee

    • Ensuring a tenant query performance with SLA

  • Support Vertica
